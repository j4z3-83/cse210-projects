Encapsulation is the practice of writing your code so that a classes variables and most of its functions cannot be directly accessed by external code.
by encapsulation your code you make it safer as external code cannot directly change any of the variables and potential break the logic. Also if you need to change a piece of code if it is encapsulated only the code that need to be changed has to be.
ensuring at all variables are private and only accessed via getter and setters.
